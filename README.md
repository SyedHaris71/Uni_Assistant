# Smart University Assistant

This is a simple AI chatbot I built for my university project. It helps students by answering common university-related questions like fees, hostel, admission, and scholarships.

The chatbot can also remember the user's name during the conversation and use it later in responses.

---

## Features

- Remembers user name during chat  
- Answers university questions like fees, hostel, admission, scholarship  
- Classifies questions into categories  
- Shows response tags like [FEE], [HOSTEL], etc.  
- Gives simple reply for unrelated questions  

---

## Tools Used

- Python  
- LangChain  
- Groq API  
- dotenv
